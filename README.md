# Dotfiles

Dotfiles for a simple Arch Linux Hyprland setup.

## Applications

- **Shell:** `bash`
- **Terminal Emulator:** `kitty`
- **System Information:** `fastfetch`
- **Status Bar:** `waybar`
- **Wallpaper Utility:** `hyprpaper`
- **Application Launcher:** `rofi-wayland`
